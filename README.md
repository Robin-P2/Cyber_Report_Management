# Cyber Report Generator

A **Django-based Cyber Report Generator** that processes uploaded rated Excel files, performs automated calculations, and generates structured analytical reports. The system uses **Django REST Framework APIs**, **JWT authentication**, and powerful data-processing libraries like **Pandas** and **NumPy** to efficiently handle Excel-based datasets.

---

## 🚀 Features

* 📂 **Excel Upload System** – Upload rated Excel files for analysis
* 📊 **Automated Data Processing** – Perform calculations using Pandas and NumPy
* 📑 **Report Generation** – Generate structured analytical reports from processed data
* 🔐 **Secure Authentication** – JWT-based authentication using SimpleJWT
* 🌐 **REST API Support** – Built with Django REST Framework for scalable APIs
* 📈 **Excel Report Export** – Generate reports using OpenPyXL and XlsxWriter

## 📂 Project Workflow

1. User uploads a **rated Excel file**
2. Backend processes the file using **Pandas and NumPy**
3. Required **calculations and transformations** are performed
4. The system generates a **structured analytical report**
5. Users can **download or view the generated report**
